# Changelog

## v5.1.0 - 2026-03-31
- 자동 파이프라인 고도화: 수집/요약/자동태깅/이슈후보/리포트 갱신을 단일 실행 흐름으로 통합.
- 운영관리 패널에서 수동 전체 파이프라인 실행 버튼 추가.
- `issue_candidates` 검토 액션(PENDING/REVIEWED/REJECTED/CONVERTED) 추가.
- 검색 품질 강화(요약/태그/카테고리/발신자 통합 검색) 및 퀵필터 추가.
- 스케줄러에 주간 뉴스레터 자동 발송 잡 추가.
- 주간 리포트 신문형 HTML 렌더러(`news_report.py`) 및 SMTP 발송(`mailer.py`) 추가.
- 운영 패널에 파이프라인 이력/스토리지 진단 표시 강화.

## v5.0.0 - 2026-03-31
- 자동 파이프라인 서비스(`PipelineService`) 추가: 수집→요약→자동태깅→이슈후보→기간리포트 자동화.
- 스케줄러를 full pipeline 기반으로 전환(자동 실행 + 수동 즉시 실행 병행).
- 운영관리 화면(`/admin/ops`) 추가: 마지막 성공/실패, 실패건수, 재시도 대기, 미요약/요약실패, 후보 미검토, 파이프라인 이력.
- `issue_candidates` + 후보 검토 흐름(`PENDING/REVIEWED/REJECTED/CONVERTED`) 추가.
- 검색 품질 강화: 제목/본문/요약/태그/카테고리/발신자 통합 검색.
